# QuickChat — Full Stack Chat App

Real-time chat application built with React, Node.js, Express, MongoDB, and Socket.IO.

## Live Demo

**https://client-gilt-rho.vercel.app**

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, Socket.IO
- **Database:** MongoDB
- **Auth:** JWT

## Run Locally

```bash
# Terminal 1 — backend
cd server
npm install
npm run server

# Terminal 2 — frontend
cd client
npm install
npm run dev
```

Open http://localhost:5173
